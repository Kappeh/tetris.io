# tetris.io
Tetris clone for internet browsers

Controls
Up or W -		Rotate falling tetmomino
Left or A -		Move falling tetrmino left
Down or S - 	Fast drop falling tetromino
Right or F - 	Move falling tetrmino right
Space - 		Hold tetrmino
Esc - 			Pause/Unpause game
